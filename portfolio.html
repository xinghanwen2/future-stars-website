Project Portfolio

This page is designed for admissions readers, coaches, recommenders, and other reviewers who need to understand the evidence behind the project.

What reviewers really want to see

  • Is Vincent truly leading the project?
  • Is this a sustained project rather than a one-time activity?
  • Does it serve real kids and families?
  • Is there a repeatable method?
  • Is there evidence of results?
  • Does the project show leadership and organization?

The evidence loop

Identify a problem → collect video and data → Vincent analyzes → assign drills → retest → parent feedback → project summary

This is the real admissions value of the project.

High-value evidence types

Founder Evidence

Founder page, work photos, analysis notes, report samples, and reflection.

Method Evidence

Standardized process, data fields, report template, and retest logic.

Impact Evidence

Anonymous player cases, parent feedback, summary numbers, and progress comparisons.
