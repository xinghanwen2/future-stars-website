# Future Stars Baseball Club - Phase 1 Website

This is a beginner-friendly static website for GitHub Pages.

## Files

- `index.html` - Home page
- `about.html` - About page
- `hitting-analysis.html` - Hitting analysis process page
- `case-studies.html` - Anonymous public case studies page
- `events.html` - Events and clinics page
- `register.html` - Registration and consent placeholder page
- `contact.html` - Contact page
- `style.css` - Website styling
- `.nojekyll` - Tells GitHub Pages not to process the site with Jekyll

## Phase 1 Goal

Build a simple public website that explains the project, shows approved anonymous case studies,
lists events, and links to parent registration/consent forms.

## Privacy Rule

Do not upload or publish real player names, photos, videos, school names, parent contact information,
or medical information without written parent approval.

## How to publish on GitHub Pages

1. Create a GitHub repository named `future-stars-baseball-club`.
2. Upload all files in this folder to the repository.
3. Go to repository `Settings`.
4. Go to `Pages`.
5. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Click `Save`.
7. Wait a few minutes.
8. Open: `https://YOUR-USERNAME.github.io/future-stars-baseball-club/`
