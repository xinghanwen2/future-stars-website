## Private Google Drive folder recommendation

Do not put private raw videos or full private reports on GitHub. Use Google Drive folders like this:

```text
Future Stars Baseball Club/
  Private Player Files/
    Player A/
      reports/
      raw-videos/
      selected-clips/
      parent-share/
```

## GitHub Pages deployment

1. Upload all files to your GitHub repository.
2. Go to **Settings > Pages**.
3. Choose **Deploy from a branch**.
4. Select branch **main** and folder **/root**.
5. Save and wait a few minutes.

## Privacy rule

Do not publish real player names, school names, parent contact information, medical information, or unapproved photos/videos.
